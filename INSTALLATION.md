# Installation

1. Clone the repository.
2. Ensure you have Node.js v22+ installed.
3. (Optional) Run `docker-compose up -d` to start local AI services (Ollama, ComfyUI).
4. Run `npm install` to install project dependencies.
5. Setup `.env` configuration file based on `.env.example`.
6. Run `npm run dev` to start the frontend and backend locally on `http://localhost:3000`.

## Production
Run `npm run build` then `npm run start`.
